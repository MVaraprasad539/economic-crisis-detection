**EcoGuard AI** is a premium, machine-learning-powered early warning system designed to detect impending economic crises. By analyzing complex non-linear relationships between macroeconomic indicators—such as GDP growth, inflation, and debt-to-GDP ratios—EcoGuard provides actionable risk scores to help policymakers and analysts preempt financial instability.

---

## 🚀 Interactive Dashboard Features
- **Real-Time Risk Engine**: Dynamic calculation of crisis probability using non-linear weighted algorithms.
- **Glassmorphism UI**: A state-of-the-art dark theme dashboard optimized for clarity and high-impact presentations.
- **Scenario Simulator**: Interactive "What-If" analysis tools to visualize how specific policy changes affect national risk.
- **Macro Analytics**: Integrated Chart.js widgets for tracking multi-variate time-series performance.
- **Historical Gallery**: Benchmarked accuracy against major historical events (2008 & 2020 recessions).

## 🛠️ Technical Architecture
### 1. Data Ingestion & Preprocessing
EcoGuard processes data sourced from the **World Bank** and **IMF**, focusing on five core "High-Impact" indicators:
- **GDP Growth**: The primary engine of economic health.
- **CPI Inflation**: Measuring purchasing power erosion.
- **Debt-to-GDP Ratio**: Assessing national fiscal sustainability.
- **Interest Rates**: The cost of liquidity and capital.
- **FX Volatility**: Currency stability in global markets.

### 2. Machine Learning Logic
The system uses an **Ensemble Learning** approach (Random Forest + Gradient Boosting) to learn from 40+ years of historical crisis labels. The high-accuracy prediction engine identifies "hidden" patterns that traditional linear Early Warning Systems (EWS) often miss.

## 💻 Installation & Local Dev
Since EcoGuard is built as a self-contained premium dashboard, it requires no backend for presentation.

1. **Clone the Repo**
2. **Launch locally**: Open `frontend/index.html` in any modern browser.
3. **Deploy**: Optimized for **Netlify** — simply drag the project folder into the Netlify dashboard.
